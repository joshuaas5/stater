# Screenshots do App

Coloque aqui os prints reais do aplicativo para a landing page.

## Arquivos necessários:

### 1. `modo-simples.png`
- Screenshot do app no **Modo Simples** (mobile)
- Tire o print no celular ou emulador
- Resolução recomendada: 390x844 (iPhone 14) ou similar
- Mostre a tela principal com saldo, entradas/saídas

### 2. `modo-completo.png`
- Screenshot do app no **Modo Completo** (desktop)
- Tire o print no navegador desktop
- Resolução recomendada: 1200x700 ou similar
- Mostre o dashboard com sidebar visível

## Como tirar os prints:

### No Chrome (Desktop):
1. Abra o app em `localhost:5173/dashboard`
2. Pressione `F12` para abrir DevTools
3. Clique em `Ctrl+Shift+P` e digite "screenshot"
4. Selecione "Capture full size screenshot" ou "Capture screenshot"

### No Celular (Android):
1. Abra o app no navegador
2. Pressione Volume Baixo + Power simultaneamente

### No Celular (iPhone):
1. Abra o app no navegador
2. Pressione Lateral + Volume Cima simultaneamente

## Dicas:
- Use dados de exemplo interessantes (não vazios)
- Esconda informações sensíveis se houver
- Prefira prints com cores vibrantes para chamar atenção
